== Markdown to Asciidoc

Use kramdown, which is better suited than pandoc as it has more AsciiDoc-sensible options.

[source,shell-session]
----
$ gem install kramdown

$ for file in ./*.md ; do
    kramdoc \
      --format=GFM \
      --output="${file%.md}.adoc" \
      --wrap=ventilate "$file"
  done
----
